## 一、规则库文本加注（S类细则）

### S-014~S-018：加注"有图/有文=已判定"
- 排放标准判定：附图/附表已有明确结论或标注"已满足标准"，视为已进行判定，不需要正文再次说明
- 判断依据：图片/表格中已标注结论（如"达标""满足"等），即视为有判定依据

### S-019：废气/废水DB44编号区分 + 甲醇HJ2.2参数说明
- **DB44/26-2001** = 广东省水污染物排放标准（废水）
- **DB44/27-2001** = 广东省大气污染物排放标准（废气）
- 甲醇HJ 2.2-2018附录D参数用于估算模型，不等于排放限值，甲醇排放标准应查GB 37823-2019

### S-022：GB18597/18599区分 + 非固废标准章节说明
- **GB 18597-2023** = 危险废物贮存污染控制标准
- **GB 18599-2020** = 一般工业固体废物贮存标准
- 1.4.2节引用的是污水处理相关内容，不属于固体废物贮存标准章节

## 二、llm_client.py prompt 优化

### 1. 新增"概念区分提醒"Section

```
## 概念区分提醒（易混淆点）
- **地下水环境承载力** ≠ 地基承载力
  - 前者：环评概念，区域地下水对污染物的容纳能力
  - 后者：岩土工程概念，地基土的强度
- **DB44/26-2001**（废水）≠ **DB44/27-2001**（废气）
- **GB 18597-2023**（危废贮存）≠ **GB 18599-2020**（一般固废贮存）
- 甲醇HJ 2.2-2018附录D参数用于估算模型，判断甲醇排放标准应以GB 37823-2019为准
```

### 2. C-020系列跨章节去重
- 同一规则在同一报告不同章节触发时，优先合并到靠后章节（结论/可行性与否章节），避免在正文论述和结论章节重复报告

### 3. 数据类缺陷5%容差标准
- 若矛盾差值 > 5%，判定为数据不一致
- 若矛盾差值 ≤ 5% 且来源可解释（如四舍五入、保留位数不同、估算误差），视为可接受的合理波动，不判为缺陷

### 4. 章节内容截断扩容
- `chapter_content[:8000]` → `chapter_content[:40000]`（40K字符）
